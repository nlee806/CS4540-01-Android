## Installation
- Required Android Studio 3.5+
- Android 5.0 or higher (Tested on 8.1)

### Clone

- Clone or download this Repository

### Setup
-  File > New > import project
-  update build/gradle scripts (if needed)
-  connect Usb Device
-  click run on device
